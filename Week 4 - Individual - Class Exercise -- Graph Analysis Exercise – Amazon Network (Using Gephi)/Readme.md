# Gephi Network Analysis: Amazon Sampled Graph

## Project Overview

This project analyzes an Amazon sampled network dataset using **Gephi**.  
The dataset was imported as an **undirected graph**, meaning that the connections between nodes do not have direction.

The analysis focuses on:

- Graph visualization
- Degree analysis
- Betweenness centrality
- PageRank
- Community detection using modularity
- Filtering and highlighting important nodes

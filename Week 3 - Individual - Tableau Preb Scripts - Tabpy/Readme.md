# Tableau Prep and TabPy Employee Data Processing

Simran-Amesar-100007050

## Overview
The workflow processes employee data from a CSV file and performs:
- IT department filtering
- Average salary calculation
- Oldest employee identification

---

## Dataset
File used:

`data/week3/employee_tableau_prep_python_sample.csv`

Dataset fields:
- Employee_ID
- Employee_Name
- Department
- Age
- Salary


---

## Python Script Features

The script performs the following:
- Filters employees from the IT department.
- Computes average salary across all employees.
- Finds the oldest employee.
- Adds new analytical columns.

---
## Output Columns

- Average_Salary_All_Employees --	Average salary across all employees
- Oldest_Employee_Name	-- Name of oldest employee
- Oldest_Employee_Department	-- Department of oldest employee
